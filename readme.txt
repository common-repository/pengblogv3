== Description ==

PengblogV3 theme is very stylish, widget supported and doesn’t require any plugin.
It's design basic on Wordpress 3.0.

Tested with IE 6/7/8, Firefox 2/3/3.5, Opera 9.6, Safari 4 and Chrome 2.0.
Valid XHTML 1.1 and CSS 3!
Best view under 1024×768 or greater.
